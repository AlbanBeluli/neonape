# Neon Ape v0.4.0

## Highlights

- Full MAGI checklist integration inside Adam
- Expanded checklist seed with passive recon, Nmap host discovery, port scanning, service enumeration, nuclei review, and web path review
- Live MAGI execution table with run/skip/done prompts and persistent checklist progress
- Passive recon checklist actions for WHOIS and DNS A-record collection
- Checklist reseed logic now preserves prior completion state

## Commands to Try

```bash
neonape adam --target example.com
neonape --show-checklist --init-only
neonape --checklist-step 2 --target example.com
neonape --checklist-step 6 --target 192.168.1.0/24
```
