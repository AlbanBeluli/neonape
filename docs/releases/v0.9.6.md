# Neon Ape v0.9.6

## Highlights

- Automatic nuclei template setup through a managed `~/.neonape/nuclei-templates` directory
- `neonape setup tools --yes` now prepares both the ffuf wordlist and nuclei templates
- Adam and the default nuclei workflow now validate templates before each nuclei run
- Safer fallback behavior when template download is unavailable

## Notes

- This release keeps all scans local-only.
- If nuclei template bootstrap cannot refresh templates, Neon Ape continues without crashing and uses whatever templates are already available.
