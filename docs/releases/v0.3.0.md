# Neon Ape v0.3.0

Second public feature release.

## Highlights

- Native `neonape obsidian` workflow with config-backed vault defaults and dry-run previews
- New `db inventory`, `db reviews`, and `db notes` views for faster triage
- Interactive shell actions for config, update, and Obsidian sync
- Built-in `neonape update` flow for install-managed copies

## Commands to Try

```bash
neonape
neonape config set obsidian_vault_path ~/Documents/Obsidian
neonape obsidian --target-note Pentests/example.com/Target.md --dry-run
neonape db inventory --target example.com
neonape db reviews --severity high
neonape update --yes
```

## Notes

- Local-only by design. No web server or exposed listener.
- Obsidian sync can now use config defaults or auto-discover the current vault.
- Dry-run mode for Obsidian sync no longer writes templates or folders.
